# erdis.kvo
Object Function Front End <br>
for NATS KV
---
provides list functions <br>
<ul>
  <li>list_append</li>
  <li>list_push</li>
  <li>list_pop</li>
  <li>list_remove_first (ocurrence)</li>
  <li>list_remove_all (ocurrences)</li>
  <li>list_valueExists - returns a list of indexes of each ocurrence</li>
</ul>
